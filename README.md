# ANITIO Institute Website v3.0

## Complete Next.js + Tailwind CSS Website

### Features Included:
- ✅ Full SEO with JSON-LD structured data, Open Graph, sitemap, robots.txt
- ✅ Online + Offline + Hybrid course modes
- ✅ Online Exam Portal (student login)
- ✅ Result Checking Portal  
- ✅ Online Payment (Razorpay + UPI + Bank Transfer)
- ✅ Franchise Page with 3 plan tiers
- ✅ About, Contact, Schedule pages
- ✅ 30+ Course pages
- ✅ News ticker, notices, WhatsApp button
- ✅ Animated hero with auto-slide
- ✅ Animated stats counters
- ✅ Google Maps integration
- ✅ All social links (WhatsApp, Instagram, Facebook, YouTube, Twitter, Telegram)
- ✅ Complete address and contact info

### Run Instructions:
```bash
# Delete any existing node_modules
rm -rf node_modules

# Install with exact pinned versions (no errors)
npm install

# Start development server
npm run dev

# Build for production
npm run build
npm run start
```
